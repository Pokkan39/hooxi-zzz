# Hooxi 绝区零剧情档案站

## 当前版本

这是一个不依赖构建工具的静态网站，包含绝区零主题路线图、章节展开/折叠、动态背景、站内音乐播放器和“小白可用”的可视化编辑器。内容按照绝区零官方 Wiki 首页可见的常用栏目整理为：主线剧情、角色剧情分支、角色专属 EP/PV、主题活动。站内条目是 Hooxi 档案的可编辑示例，不代表 Hooxi 的真实投稿。

## 小白编辑流程

1. 打开网站，点击右上角黄色 `✦`，进入“站点编辑器”。
2. **外观**：修改主标题、首页介绍、背景图片 URL、透明度和动态效果开关；点击“应用外观”即可立即生效。
3. **内容**：修改已有剧情卡片的分类和标题，或点击“新增剧情卡片”。保存后路线图立即刷新。视频地址可在 `app.js` 的数据中继续补充，或后续扩展编辑字段。
4. **歌单**：填写歌曲名称和你有权使用的 MP3/OGG 地址；保存后底部播放器可以播放、切换、调节音量。
5. **导出配置**：把当前所有内容保存为 JSON 备份；换电脑时用“导入配置”恢复。需要回到初始状态时点击“恢复默认”。

配置会保存在当前浏览器的 `localStorage` 中，不需要数据库。若清理浏览器数据，先导出配置备份。

## 音乐与自动播放说明

播放器会在页面加载时尝试自动播放，但现代浏览器通常会阻止带声音的自动播放，这是浏览器安全策略，无法通过网页可靠绕过。出现提示时点击底部播放按钮即可启用。请只使用自己创作、获得授权或明确允许使用的音频地址。

## 多页面与排序

网站现在提供四个独立档案页：主线时间轴、支线/角色故事、代理人 EP/PV、主题活动。每个页面的卡片都可以用鼠标拖动重新排序，排序保存在当前浏览器；`data.js` 是默认数据源。

## 音频文件发布

在站内编辑器的“歌单”标签中选择本地音频，可以立即在当前浏览器试听。若要让 GitHub Pages 的所有访客都能听到：将授权音频复制到仓库 `assets/audio/`，点击“生成发布清单 playlist.json”，把清单中的地址改成 `assets/audio/文件名.mp3`，然后把音频和 JSON 一起提交到 GitHub。纯静态 GitHub Pages 不能让网页直接写入仓库，因此不能在公开网页中放 GitHub Token。


在项目目录执行：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080/`。
